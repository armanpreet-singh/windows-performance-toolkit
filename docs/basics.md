# Basic Windows Configuration

These optimizations are safe for most users and should be completed before applying advanced tweaks.

---

# Overview

The **Basic** section focuses on improving Windows responsiveness without making major system modifications.

These settings are recommended before applying registry tweaks or advanced optimizations.

---

# Included Optimizations

- Display Settings
- Notifications
- Startup Applications
- Visual Effects
- Mouse Settings
- Game Mode
- Power Plans
- Memory Compression
- System Configuration

---

# Recommended Order

1. Configure Display
2. Disable unnecessary Notifications
3. Disable unnecessary Startup Apps
4. Adjust Visual Effects
5. Configure Mouse Settings
6. Enable appropriate Power Plan
7. Review Memory Compression
8. Restart Windows
