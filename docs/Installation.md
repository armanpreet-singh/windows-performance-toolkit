# Installation Guide

Welcome to the Windows Performance Toolkit.

This guide explains how to use the toolkit safely.

---

# Before You Begin

Before applying any optimization:

- Create a Windows Restore Point.
- Backup important files.
- Backup your Registry.
- Read the documentation for every tweak.
- Apply one tweak at a time.

> Never apply every tweak at once.

---

# Requirements

Supported Operating Systems:

- Windows 10
- Windows 11

Administrator privileges are required for many scripts and registry changes.

---

# Recommended Workflow

For the safest experience, follow this order:

## 1. Create a Restore Point

Always create a system restore point before making changes.

Documentation:

- Restore Guide

---

## 2. Install Required Applications

Some optimizations require third-party utilities.

Examples:

- AutoRuns
- QuickCPU
- MSI Utility
- Device Cleanup

Each application has its own documentation page.

---

## 3. Apply Basic Tweaks

Start with low-risk optimizations before moving to advanced configurations.

---

## 4. Apply Registry Tweaks

Review every registry file before importing it.

Restart Windows after applying registry changes unless otherwise specified.

---

## 5. Configure Networking

Apply network optimizations only if you understand their purpose.

---

## 6. Apply Advanced Tweaks

Advanced tweaks should only be used after reading the documentation.

---

## 7. Verify System Stability

After each change:

- Restart Windows
- Check Device Manager
- Test applications
- Monitor temperatures
- Confirm system stability
