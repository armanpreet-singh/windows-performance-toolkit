# 🚀 Windows Performance Toolkit

> A curated collection of Windows performance tweaks, cleanup scripts, registry optimizations, power plans, networking utilities, and system configuration guides.

---

## 📖 Overview

Windows Performance Toolkit is an open-source collection of carefully organized tools, scripts, registry tweaks, and documentation designed to help users optimize Windows performance safely and systematically.

Unlike many tweak packs, this repository focuses on **understanding each optimization** before applying it. Every tweak will include documentation explaining:

- What it does
- Why it exists
- When to use it
- Possible risks
- How to revert it

---

## ⚠️ Disclaimer

This repository contains system-level tweaks that may affect Windows behavior.

Always:

- Create a Restore Point
- Read the documentation
- Test one tweak at a time
- Backup important data

Use everything at your own risk.
