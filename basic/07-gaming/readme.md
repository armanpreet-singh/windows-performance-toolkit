# Game Mode

## Overview

Windows Game Mode is a built-in feature that prioritizes game processes by reducing background activity and scheduling more system resources for the active game.

While Microsoft recommends Game Mode for gaming, its effectiveness varies depending on hardware, Windows version, and the games being played.

This module provides quick access to the Game Mode settings so users can enable or disable it and compare performance.

---

# Included File

| File | Description |
|------|-------------|
| Game Mode.url | Opens the Windows Game Mode settings page |

---

# What Does Game Mode Do?

When enabled, Windows attempts to:

- Prioritize the running game
- Reduce background Windows Update activity
- Reduce some background notifications
- Allocate more CPU resources to the active game

---

# Why Disable Game Mode?

Some users report better consistency with Game Mode disabled, especially on:

- High-end desktop PCs
- Competitive FPS games
- Systems already optimized for gaming

Possible benefits include:

- More consistent frame times
- Lower input latency (depending on the game)
- Fewer scheduling conflicts

> Results vary between systems. Always test before deciding.

---

# How to Disable Game Mode

1. Open **Game Mode** using the included shortcut.
2. Toggle **Game Mode** to **Off**.
3. Restart the game.
4. Compare performance and input responsiveness.

---

# Should You Disable It?

You may want to test Game Mode **Off** if:

- You play competitive games.
- Your system already has minimal background processes.
- You have a modern multi-core CPU.
- You notice inconsistent frame pacing.

---

# Should You Keep It Enabled?

Keeping Game Mode enabled may help if:

- You have lower-end hardware.
- Many applications run in the background while gaming.
- You experience stuttering caused by background tasks.

---

# Best Practices

- Benchmark before and after changing the setting.
- Restart the game after modifying Game Mode.
- Change only one optimization at a time.
- Record FPS and frame times when testing.

# Notes

Game Mode is not guaranteed to improve gaming performance.

The best setting depends on your hardware, drivers, Windows version, and the specific games you play.

Always benchmark your own system before deciding whether to keep it enabled or disabled.
