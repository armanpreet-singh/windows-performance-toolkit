# Visual Effects

## Overview

Windows includes animations, shadows, transparency, and other visual effects that improve appearance but may consume additional system resources.

This section explains how to adjust these settings for a balance between performance and usability.

---

## Included File

| File | Purpose |
|------|---------|
| Visual Effects.url | Opens the Windows Performance Options dialog |

---

## Why Optimize Visual Effects?

Reducing unnecessary visual effects can:

- Improve UI responsiveness
- Reduce CPU usage during UI rendering
- Improve performance on older hardware
- Make Windows feel more responsive

---

## Recommended Settings

For maximum performance:

- Disable window animations
- Disable fade effects
- Disable menu animations
- Disable shadows under windows
- Disable transparent selection rectangle

Keep enabled:

- ✔ Smooth edges of screen fonts
- ✔ Show thumbnails instead of icons

These provide a better user experience without significantly affecting performance.

---

## How to Apply

1. Open **Visual Effects**.
2. Select **Custom**.
3. Disable unnecessary animations.
4. Keep font smoothing enabled.
5. Click **Apply**.
6. Restart Explorer or restart Windows if required.

---

## Risk Level

🟢 Low

These settings only affect the Windows interface and can be reverted at any time.
