# Display Configuration

## Overview

Display settings have a small but noticeable impact on the overall Windows user experience.

This section focuses on configuring display-related settings that improve usability and reduce unnecessary visual overhead.

---

## Included File

| File | Purpose |
|------|---------|
| display.url | Opens the Windows Display Settings page |

---

## Why is this included?

Many Windows optimizations begin by configuring:

- Display Scaling
- Refresh Rate
- Resolution
- Night Light
- HDR
- Multiple Displays

Keeping this shortcut provides quick access to these settings.

---

## How to Use

1. Double-click `display.url`.
2. Windows will open **Settings → System → Display**.
3. Configure the settings according to your hardware.

---

## Recommended Settings

These depend on your monitor.

Examples:

- Native Resolution
- Highest Supported Refresh Rate
- Correct Scaling
- HDR (if supported)
