# Audio Configuration

## Overview

This module provides quick access to the classic Windows Sound Control Panel, allowing users to configure playback devices, recording devices, and audio enhancements.

Proper audio configuration can improve gaming, communication, and multimedia experiences.

> ⚠️ Audio settings do not directly increase FPS, but incorrect settings may increase latency or reduce sound quality.

---

# Included File

| File | Description |
|------|-------------|
| Sound Settings.lnk | Opens the classic Windows Sound Control Panel |

---

# Why is this included?

Many important audio options are still located in the classic Sound Control Panel rather than the modern Windows Settings app.

This includes:

- Default playback device
- Recording devices
- Audio enhancements
- Sample rate configuration
- Exclusive Mode
- Communications settings

---

# Recommended Configuration

## Playback Device

- Select the correct default output device.
- Disable unused playback devices if desired.

---

## Recording Device

- Select the microphone you actually use.
- Disable unused microphones to reduce clutter.

---

## Communications Tab

Set:

```
Do Nothing
```

This prevents Windows from automatically lowering game or media volume during voice calls.

---

## Audio Enhancements

If available:

- Disable unnecessary audio enhancements.
- Test sound quality after making changes.

Some manufacturers provide enhancements that may improve or degrade audio depending on the hardware.

---

## Sample Rate

For most users:

```
24-bit
48000 Hz
```

is a good default for gaming and multimedia.

Only change this if your audio hardware supports it.

---

# Best Practices

- Install the latest audio drivers.
- Restart applications after changing audio settings.
- Test microphone and speakers after making modifications.
- Use wired audio devices for the lowest latency when possible.
