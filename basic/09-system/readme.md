# System Configuration

## Overview

This module contains Windows system configuration shortcuts that provide quick access to commonly used performance-related settings.

---

# Included Files

| File | Description |
|------|-------------|
| System Configuration.lnk | Opens the System Configuration utility (msconfig) |
| Performance Options.lnk | Opens the Windows Performance Options dialog |
| Notifications.url | Opens the Windows Notifications settings |

---

# Performance Options

Recommended:

- Adjust for best performance

Optionally enable:

- Smooth edges of screen fonts
- Show thumbnails instead of icons

---

# Notifications

Recommended:

- Disable unnecessary notifications
- Disable notification sounds if desired
- Enable Do Not Disturb while gaming

---

# System Configuration (Boot Advanced Options)

⚠️ **Do NOT manually set:**

- Number of processors
- Maximum memory

Windows automatically detects and uses all available CPU cores and installed memory.

These options are intended for debugging and troubleshooting, **not performance optimization**.

Leave both options **unchecked** unless you have a specific troubleshooting requirement.

---

# Risk Level

🟢 Low

These settings are safe when used correctly.

Avoid changing boot configuration options unless you understand their purpose.
