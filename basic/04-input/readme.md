# Input Optimization

## Overview

Input optimization focuses on improving the responsiveness of mouse and keyboard interactions.

Although these tweaks do not increase FPS, they can improve the overall desktop and gaming experience by reducing unnecessary processing delays and ensuring consistent input behavior.

---

## Included Files

| File | Purpose |
|------|---------|
| Mouse | Mouse-related settings |
| Keyboard | Keyboard-related settings |

---

## Mouse Configuration

Recommended settings:

- Pointer Speed: 6/11
- Disable "Enhance Pointer Precision" (unless required)
- Use native DPI from your mouse software

### Why?

Disabling pointer acceleration provides consistent cursor movement and improves muscle memory.

---

## Keyboard Configuration

Recommended settings:

- Repeat Delay: Short
- Repeat Rate: Fast

These settings improve typing responsiveness.

---

## Best Practices

- Use the latest mouse drivers.
- Use the latest keyboard firmware if available.
- Prefer wired devices for competitive gaming.
- Test one change at a time.

---

## Risk Level

🟢 Low

These settings are fully reversible through Windows Settings.
