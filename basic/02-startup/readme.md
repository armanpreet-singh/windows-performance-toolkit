# Startup Applications

## Overview

Startup applications are programs that automatically launch when Windows starts.

Reducing unnecessary startup applications can improve:

- Boot time
- Memory usage
- CPU usage
- Overall system responsiveness

---

## Included File

| File | Purpose |
|------|---------|
| Startup Apps.url | Opens Windows Startup Apps settings |

---

## Why Optimize Startup Apps?

Many applications automatically add themselves to startup.

Examples include:

- Discord
- Spotify
- Steam
- Epic Games Launcher
- Adobe Creative Cloud
- Microsoft Teams

Disabling unnecessary startup apps reduces background resource usage.

---

## How to Use

1. Open **Startup Apps**.
2. Sort applications by **Startup Impact**.
3. Disable applications you don't need immediately after login.
4. Restart Windows.

---

## Recommended

Keep Enabled:

- Windows Security
- GPU Drivers
- Audio Drivers
- Touchpad Drivers
- Hardware Utilities

Consider Disabling:

- Discord
- Spotify
- Steam
- Adobe Updater
- Teams (if not required)

---

## Risk Level

🟢 Low

Disabling startup does **not** uninstall an application.

You can re-enable it at any time.
